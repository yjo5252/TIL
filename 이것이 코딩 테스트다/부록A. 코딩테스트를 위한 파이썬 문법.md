## 02. 조건문 

#### 0. 활용
* 리스트에 있는 원소의 값을 또다른 리스트로 만든다
```python
// 리스트 컴프리헨션, 조건문 사용
a = [1, 2, 3, 4, 5, 5, 5]
remove_set={3,5}

result = [i for i in a if not in remove_set]

print(result) // [1, 2, 4]
```
